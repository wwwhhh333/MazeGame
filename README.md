# 数据结构课程作业：迷宫游戏

## 游戏概述
程序开始运行时显示一个迷宫地图，迷宫中央有一只老鼠，迷宫的右下方有一个粮仓。游戏的任务是使用键盘上的方向键操纵老鼠在规定的时间内走到粮仓处。

## 设计要求

### 1. 老鼠控制
- 老鼠形象可辨认
- 使用键盘方向键控制老鼠上下左右移动

### 2. 结果检测
- 检查老鼠是否在规定时间内到达粮仓
- 成功到达：提示成功
- 未到达：提示失败

### 3. 迷宫编辑功能
- 可修改当前迷宫
- 支持的编辑操作：
  - 墙变路
  - 路变墙

### 4. 路径分析
- 找出走出迷宫的所有可能路径
- 计算并显示最短路径

## 附加说明
- 游戏界面应清晰直观
- 用户交互友好
- 路径查找算法需高效准确
